# Intelligent Sales Co-Pilot

AI-powered real-time sales assistance — pre-call research, live conversation intelligence, and post-call synthesis.

**QuackHacks '26 | ADP Challenge: "Hack the Sales Journey with AI"**

## Quick Start

**Backend** (Python/FastAPI):
```bash
cd backend
pip install -r requirements.txt
cp .env.example .env  # fill in your keys
uvicorn app.main:app --reload --port 8000
```

**Frontend** (Next.js):
```bash
cd frontend
npm install
npm run dev
```
